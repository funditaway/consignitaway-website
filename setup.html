  1. npx wrangler login
  2. npx wrangler kv namespace create CONSIGNED_DATA --config wrangler-worker.toml
  3. Paste KV id into wrangler-worker.toml
  4. npm run deploy:worker
  5. Paste Worker URL above and save
  6. npx wrangler secret put SQUARE_ACCESS_TOKEN --config wrangler-worker.toml
